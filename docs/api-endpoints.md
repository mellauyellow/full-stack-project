# API Endpoints

## HTML API

### Root

- `GET /` - loads React web app

## JSON API

### Users

- `POST /api/users`
- `PATCH /api/users`

### Session

- `POST /api/session`
- `DELETE /api/session`
- `GET /api/session`

### Regions

- `GET /api/regions`

### Neighborhoods

- `GET /api/neighborhoods`
- `GET /api/neighborhoods/:id`

### Reviews

- `GET /api/reviews`
- `GET /api/reviews/:id`
- `POST /api/reviews`
- `PATCH /api/reviews/:id`
- `DELETE /api/reviews/:id`
